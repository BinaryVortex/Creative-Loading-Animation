# Creative Loading Animation

Creative Loading Animation built using only HTML and CSS — a lightweight, customizable loader that you can drop into any project.

## Preview

![Preview of the loader](https://raw.githubusercontent.com/BinaryVortex/Creative-Loading-Animation/main/Screenshot%202024-05-03%20183950.png)

## Features

- Pure HTML & CSS (no JavaScript)
- Lightweight and easy to integrate
- Easy to customize color, size, and animation speed
- Great for demos, landing pages, or placeholders during content loading

## Files

- `index.html` — minimal demo page showing the loader
- `style.css` — styles and animation rules
- `Screenshot 2024-05-03 183950.png` — preview image shown above

## Usage

1. Clone or download the repo:
   ```bash
   git clone https://github.com/BinaryVortex/Creative-Loading-Animation.git
   ```
2. Open `index.html` in your browser to see the demo.
3. To embed the loader in your project, copy the relevant HTML snippet from `index.html` and the CSS from `style.css` into your project files.

Example (basic):
```html
<link rel="stylesheet" href="style.css">

<!-- loader HTML -->
<div class="loader">
  <!-- loader structure from index.html -->
</div>
```

## Customization

Open `style.css` and tweak the following to match your project:

- Colors: change any color values to your brand or preference
- Size: modify width/height or container scale
- Animation speed: adjust `animation-duration` values to slow down or speed up the motion

(The CSS is small and documented — it should be straightforward to find the rules to change.)
